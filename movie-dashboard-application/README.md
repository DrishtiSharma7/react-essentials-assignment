# Movie Explorer

A react application that allow user to browse movies, search by tittle, filter by genre, and save favorites.

## WHY I BUILD THIS

I create this project to strengthen my understanding of React hooks such as useState and useMemo and to practice building interactive user interfaces.

## FEATURES

- Search movies by title
- Mark/Unmark favorites
- Responsive design
- Favorites saved using localStorage

# TECH STACK

- React
- CSS3
- JavaScript
